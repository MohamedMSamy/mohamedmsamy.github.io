001 Current · Spring 2026

Modular Omni-Wheel Mobile Manipulator

Designing and building a fully integrated mobile manipulator for the Spring 2026 Mechatronics Omni-Challenge at Ain Shams University. The robot combines a holonomic omni-wheel base with a modular robotic arm and gripper, operating across two modes: manual wireless teleoperation for track navigation and a fully autonomous pick-and-place phase using QR code detection for color-based cube sorting into onboard bins and delivery to matching drop-off zones.

Each module is engineered as a standalone, plug-and-play subsystem with its own controller, power distribution, and standardized mechanical and electrical interfaces. The system integrates wheel encoder odometry, IMU-based heading stabilization, and a camera vision pipeline for lane detection and object classification.
