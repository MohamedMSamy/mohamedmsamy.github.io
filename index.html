Modular Omni-Wheel Mobile Manipulator

Built a fully integrated mobile manipulator for the Spring 2026 Mechatronics Omni-Challenge at Ain Shams University, placing 2nd overall with a perfect task score — 6 out of 6 tasks completed, zero penalties.

The robot combines a holonomic omni-wheel base with a modular 5-DOF arm and gripper, operating across two modes: manual wireless teleoperation for track navigation and a fully autonomous pick-and-place phase using QR code detection for colour-coded cube sorting and delivery to matching drop-off zones.

The system runs on three ESP32 controllers (base + arm + vision) communicating over ESP-NOW, with the ESP32-CAM and quirc library handling QR decoding and a WebSocket layer coordinating real-time inter-module state.
